Unicode2Bijoy

PHP class to convert Unicode to Bijoy ANSI. There are plenty of PHP classes to convert Bijoy to Unicode but I didn't find any class to convert Unicode to Bijoy ANSI. That is why I coded this simple yet useful script :)

Install via composer

composer require mirazmac/unicode2bijoy

Example

$str='জানার আছে অনেক কিছু!';
echo mirazmac\Unicode2Bijoy::convert($str);
