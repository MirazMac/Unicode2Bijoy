# Unicode2Bijoy
PHP class to convert unicode to Bijoy ANSI.
### Example

```php
require_once __DIR__.'/Unicode2Bijoy.class.php';
$str='????? ??? ???? ????!';
echo Unicode2Bijoy::convert($str);
```
